# Responsible AI Strategy Framework

AI를 적극적으로 활용하되, 책임 규모가 커질수록 사람이 구조와 맥락과 최종 검증을 가져야 한다는 관점을 정리하는 프로젝트다.

## 최종 산출물

- GitHub Pages entry: `docs/index.html`
- One-page framework: `docs/ai-strategy-framework.html`

## 문서 구조

- `notes/big-flow.md`: 전체 논리 흐름
- `notes/inbox/`: 원문에 가까운 브레인스토밍 기록
- `notes/nodes/`: 개별 개념, 비유, 근거 정리
- `notes/questions/`: 말하기용 Q&A
- `AGENTS.md`: 프로젝트 목적과 작업 규칙

## 핵심 문장
